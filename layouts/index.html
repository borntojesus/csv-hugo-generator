📚 How to Add New Scrapers (Developer Guide)

This site uses Hugo Content Adapters to automatically generate pages from CSV data!

🔧 Quick Setup:

  1. Edit CSV File: assets/gen-posts.csv
  2. Add New Row: Fill in all 8 columns (slug, title, description, etc.)
  3. Build Site: Run hugo or hugo server
  4. Done! New page available at /scrapers/{slug}/

📋 CSV Column Reference:

slug:
URL path (e.g., "google-maps-scraper")
title_tag_goes_here:
HTML title tag for SEO
meta_description_goes_here:
Meta description for SEO
hero_section_heading:
Main heading on page
hero_section_intro:
Intro paragraph text
benefit_variable_1,2,3:
Three key benefit points
